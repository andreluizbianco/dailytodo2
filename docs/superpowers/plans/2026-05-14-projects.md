# Projects Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Projects view that groups notes under project stickies while preserving the existing Notes workflow.

**Architecture:** Projects are persisted alongside todos and each `Todo` may reference one `projectId`. `App` owns selection state for projects, notes, and note source; `ProjectList` renders project stickies plus indented child notes from Notes, Archive, and Calendar; `TodoNoteColumn` renders either a project note or a normal note depending on active selection.

**Tech Stack:** React Native/Expo, AsyncStorage, TypeScript, existing sticky-note components and theme helpers.
